# Irodori — What To Do

A self-assessment questionnaire for the [Irodori](https://www.irodori.jpf.go.jp/)
Japanese textbooks: gauge, lesson by lesson, what you can already do, and get a
plan of which lessons to **skip**, **review**, or **study**.

## Build

```bash
uv run irodori
```

This regenerates the static site into `docs/` (served via GitHub Pages).

See [CLAUDE.md](CLAUDE.md) for the full pipeline and project layout.
